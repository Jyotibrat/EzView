Features

  • Live Markdown Editing: Type your Markdown and preview the formatted output instantly.
  • Customizable Themes: Switch between themes to suit your preferences.
  • Download as HTML: Save your Markdown preview as a downloadable HTML file.
  • Beginner-Friendly Codebase: Ideal for beginner developers looking to contribute.

Technologies Used

MarkView is built using the following technologies:

  • HTML5
  • CSS3
  • JavaScript
  • Marked JS Library
  • GitHub Pages for deployment
