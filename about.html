Markdown Previewer

About MarkView

Overview

MarkView is a simple yet powerful Markdown Previewer designed to help users write and preview Markdown content in real-time. It provides a seamless writing experience by combining live editing with customizable themes and export options. This project is a part of the Android Club Winter of Code 2024 initiative and is open to contributions from developers of all levels.

How to use ?

MarkView is easy to use. Simply type your Markdown content in the input box on the left and see the formatted output in the preview section on the right. You can switch between different themes, download the preview as an HTML file, and adjust the font size to suit your preferences. MarkView is designed to be beginner-friendly and is ideal for anyone looking to write and preview Markdown content effortlessly.

Features

  • Live Markdown Editing: Type your Markdown and preview the formatted output instantly.
  • Customizable Themes: Switch between themes to suit your preferences.
  • Download as HTML: Save your Markdown preview as a downloadable HTML file.
  • Beginner-Friendly Codebase: Ideal for beginner developers looking to contribute.

Technologies Used

MarkView is built using the following technologies:

  • HTML5
  • CSS3
  • JavaScript
  • Marked JS Library
  • GitHub Pages for deployment

Contribution

MarkView is an open-source project aimed at fostering contributions from developers. Whether you are a beginner or an experienced developer, we welcome you to contribute and help improve this project. Visit our GitHub repository to get started.
